# Rocq学習カリキュラム — ソフトウェア検証入門

プログラミング経験者がRocq (旧Coq) をゼロから学び、ソフトウェアの正しさを証明できるようになるためのカリキュラム。Software Foundationsの流れに沿いつつ、実践的な検証スキルを身につける。

## 難易度カーブ

- **01-02**: 入門 — Rocqの世界に慣れる
- **03-05**: 基礎 — 証明の基本技法を習得
- **06-07**: 中級 — 論理学の形式化を理解
- **08-10**: 応用 — プログラムを形式化して性質を証明
- **11-12**: 実践 — ソフトウェア検証の本格的な手法

## 各トピックの構成

```
XX-topic-name/
├── main.v              ← サンプルコード + 演習の雛形
└── README.md           ← 概念説明 + 演習
```

### README.md の構成

1. **概要** — トピックの簡潔な説明
2. **背景知識** — プログラミング経験者が理解しやすいように既知の概念と対比
3. **コード解説** — `main.v` のコードを段階的に解説
4. **演習** — 理解を確認するための課題 (段階的に難易度を上げる)
5. **まとめ** — 学んだポイントの箇条書き

## 進め方

1. 各トピックのREADME.mdを読む
2. `main.v` をRocqIDEで開き、対話的に証明を進める
3. 演習に取り組む
4. 理解できたら次のトピックへ進む
