# daily — 毎朝のテック＆セキュリティ ダイジェスト

毎朝 Claude Code で `/daily` を実行すると、その日の `YYYY-MM-DD.md` がここに生成される。

- **対象分野**: 自分の技術スタック (Rust/Go/AWS) / 脆弱性・セキュリティ / 映像配信・メディア / 一般テックニュース
- **形式**: 英語の見出し＋日本語要約、ソースリンク付き、脆弱性は深刻度明記
- **深掘り**: 生成後、そのままセッションで「この脆弱性詳しく」などと続けて調べられる

## 使い方

```
/daily                  # 今日のダイジェストを生成
/daily WebAssembly      # 追加で重点的に拾うキーワードを指定
```

コマンド定義は `.claude/commands/daily.md`。重点分野や形式を変えたいときはそのファイルを編集する。

> 対話セッション内で動くため、Maxプランの対話枠で完結し、自動実行(routine)の従量課金(Agent SDKクレジット)には影響しない。
