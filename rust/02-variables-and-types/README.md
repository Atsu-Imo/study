# 02 - 変数と型

## 概要

Rustの変数宣言（`let` / `let mut`）、型推論、シャドウイング、基本的なデータ型を学ぶ。Goとの最大の違いは**変数がデフォルトで不変**であること。

## Goとの比較

| 概念 | Go | Rust |
|---|---|---|
| 変数宣言 | `var x int = 5` / `x := 5` | `let x: i32 = 5` / `let x = 5` |
| 可変性 | 常にmutable | デフォルトimmutable。`let mut` で可変 |
| 定数 | `const x = 5`（型推論あり） | `const X: i32 = 5`（型注釈必須） |
| 型推論 | `:=` で推論 | `let` で推論 |
| シャドウイング | 同スコープでは不可 | 同スコープで可能（型も変更可） |
| タプル | なし（多値返却はある） | `(i32, f64, bool)` のように使える |
| 配列 | `[5]int`（固定長） | `[i32; 5]`（固定長） |
| スライス | `[]int`（可変長） | `&[i32]`（後のトピックで学ぶ） |

### Goとの主な違い

- **不変がデフォルト**: Goでは全ての変数が再代入可能。Rustでは `mut` を明示的に付けないと再代入できない。これにより意図しない変更を防ぐ
- **シャドウイング**: Goでは同じスコープで同名の変数を再宣言できない。Rustでは `let` で同名の変数を新しく宣言でき、型も変えられる
- **タプル型**: Goには独立したタプル型がない（関数の多値返却はあるが型としては存在しない）。Rustではタプルはファーストクラスの型
- **ユニット型 `()`**: Goの関数で戻り値なしは単に省略するが、Rustでは暗黙的に `()` を返す

## 基本型一覧

### 整数型

| Rust | Go | サイズ |
|---|---|---|
| `i8` | `int8` | 8bit 符号付き |
| `i16` | `int16` | 16bit 符号付き |
| `i32` | `int32` | 32bit 符号付き |
| `i64` | `int64` | 64bit 符号付き |
| `i128` | なし | 128bit 符号付き |
| `isize` | `int` | ポインタサイズ |
| `u8` | `uint8` / `byte` | 8bit 符号なし |
| `u16` | `uint16` | 16bit 符号なし |
| `u32` | `uint32` | 32bit 符号なし |
| `u64` | `uint64` | 64bit 符号なし |
| `u128` | なし | 128bit 符号なし |
| `usize` | `uint` / `uintptr` | ポインタサイズ |

### その他の基本型

| Rust | Go | 説明 |
|---|---|---|
| `f32` | `float32` | 32bit 浮動小数点 |
| `f64` | `float64` | 64bit 浮動小数点 |
| `bool` | `bool` | 真偽値 |
| `char` | `rune` (`int32`) | Unicode 1文字（Rustは4バイト） |
| `&str` | `string` | 文字列スライス（不変） |
| `String` | `string` | ヒープ上の文字列（可変） |
| `()` | なし | ユニット型（値なし） |

## コード解説

### 不変と可変

```rust
let x = 5;       // 不変（再代入不可）
let mut y = 10;   // 可変（再代入可能）
y = 20;           // OK
// x = 10;        // コンパイルエラー
```

```go
// Goでは常にmutable
x := 5
x = 10  // OK（Goでは常に再代入可能）
```

### シャドウイング

```rust
let s = "123";      // &str型
let s = s.len();    // usize型に変わる（新しい変数）
let s = s * 2;      // 値の計算
// 型が変わっても同じ名前を再利用できる
```

これは `mut` とは違う。`mut` は同じ変数の値を変えるが型は変えられない。シャドウイングは新しい変数を作るので型も変えられる。

### タプル

```rust
let tup = (500, 6.4, true);

// 分割代入（destructuring）
let (x, y, z) = tup;

// インデックスアクセス（0始まり、ドット記法）
println!("{}", tup.0);  // 500
```

```go
// Goにはタプル型がない。多値返却は似ているが型としては存在しない
// x, y := someFunc()  // 多値返却
```

### 配列

```rust
let arr = [1, 2, 3, 4, 5];   // 型は [i32; 5]
let zeros = [0; 5];           // [0, 0, 0, 0, 0]
println!("{}", arr[0]);       // インデックスアクセス
println!("{}", arr.len());    // 長さ
```

```go
arr := [5]int{1, 2, 3, 4, 5}  // Goの固定長配列
```

## まとめ

- 変数は `let` で宣言、**デフォルトで不変**。`let mut` で可変にする
- 型推論があるが、`let x: i32 = 5` のように明示もできる
- **シャドウイング**で同名変数を再宣言可能（型も変更可）。`mut` とは別の概念
- 整数は `i32` がデフォルト、浮動小数点は `f64` がデフォルト
- **タプル**は異なる型をまとめる。Goにはない型
- **配列**は固定長。`[型; 長さ]` で表す
- **ユニット型 `()`** は「値なし」を表す
